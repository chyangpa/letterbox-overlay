# Letterbox Overlay

## 개요
활성 창 뒤에 전체화면 검정 오버레이를 배치하여 레터박스 효과를 만드는 프로그램.
캡처 없이 순수 오버레이 방식으로 딜레이 없음.

## 기술 스택
- Python
- Windows API (ctypes/pywin32)
- tkinter (오버레이 창)

## 핵심 동작

### 토글
- `Ctrl+Alt+D`로 ON/OFF
- ON: 현재 포커스된 창을 대상으로 지정, 검정 오버레이 활성화
- OFF: 오버레이 해제, 대상 해제

### 오버레이 방식
- 전체화면 검정 창을 대상 창 바로 뒤(Z-order)에 배치
- 창 위치/크기 계산 불필요 — 대상 창이 알아서 가림
- `SetWindowPos(black_hwnd, target_hwnd, ...)` 로 Z-order 유지

### 포커스 연동
- 대상 창이 포커스를 잃으면 → 오버레이 숨김
- 대상 창이 다시 포커스 받으면 → 오버레이 표시 + Z-order 재배치
- `SetWinEventHook`으로 포커스 변경 이벤트 감지

### 대상 범위
- 창모드 / 보더리스 창만 지원
- Exclusive fullscreen은 대상 아님

### 모니터
- 대상 창이 있는 모니터만 덮음

### 트레이
- 시스템 트레이 상주
- 트레이 아이콘으로 상태 확인 / 종료

## 프로토타입 단계
- 전체화면 대신 **작은 검정 창**으로 먼저 동작 검증
- Z-order 배치, 포커스 연동, 단축키 토글 확인 후 전체화면으로 확장

## 용어
- Passthrough: LSFG(Lossless Scaling)에서 프레임 생성/스케일링 모두 끈 모드
- 이 프로그램의 원래 목적: Passthrough 모드에서 컨텐츠를 중앙 정렬하고 나머지를 검정으로 채우는 것
